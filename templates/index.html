Mega Polls

About Website

This website provides royalty free polls creation. Make polls, share with friends and make decisions together. To create poll use appbar at the top of webstie. To share with friends just copy created poll link. That's it! Really simple, isn't it? Seems like it's something like google forms, but more wierd)

What can I use website for?

This website was created for social networks like WhatsApp or Discord, where you can't directly create a poll(like in Telegram) and send it to your friends. But you can use this website, create poll, copy link and after some time view results of the poll.

Website development

Technical task

Create website that will provide simple polls creation with local database, pretty design and simple architecture that everyone will understand.
There will be some pages:
  • Main page that show information about website.
  • Account creation page.
  • Account page that shows some info to user and let log out account.
  • Poll page itself that will show poll options to users who didn't vote and results of poll to users who voted already.
Beautiful design will be provided mostly with JS with canvas element that will create beautiful particles effect, some modern CSS styles(with glowing and opacity) and also with some simple features like fonts.

Explanation note

This website provides simple polls creation. Here I'll tell you about website structure and technologies this website uses.

Structure.

  • Main page where you can read information about website and navigate to other pages(main route).
  • Account creation/login page where you can create account or log in it(create_account route).
  • Polls creation page where... I don't need to explain what can you do there(create_poll route).
  • Account page where you can log out account(account route).
  • Vote in poll page where you can vote in poll and if you already voted, you can view results
    (poll?poll=poll.id route).

    • Technologies.

    Website uses Python 3 with flask framework as backend(mostly database management).
    Libraries:
  • Flask(to create website itself)
  • sqlalchemy(as database management)
  • sqlite3(as alternative database management)
  • built-in libraries(like sys, ast)
    • HTML, CSS, JS as frontend(biggest part of code).
    Modifications:
  • Bootstrap icons for html
  • sweetalert2 to make beautiful alerts
    • Heroku as hosting.
    git as version control system.
